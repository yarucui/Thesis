# Master Thesis: LLM-Human Alignment Gap

## Core argument
The alignment gap between LLM and human responses is misattributed.
Framing/persona is statistically detectable but practically negligible.
The real drivers are: distribution-aware sampling methods (Δ ≈ 0.23),
task structure (Δ ≈ 0.54), and domain priors (Δ ≈ 0.18).

## Writing style
- Academic but direct. No "it should be noted that"-style padding.
- Active voice where possible.
- Past tense for methods/results, present tense for established facts.
- One idea per paragraph. First sentence = topic sentence.
- No more than 3 sentences before a citation in related work.

## Structure conventions
- Sections: \section{Sentence case} (not Title Case)
- Figures: always use \cref{}, never "Figure X"
- Numbers: 1-9 spelled out, ≥10 numerals; always numerals before units
