# CLAUDE.md — Thesis Writing Rules

## Core argument (LOCKED — do not modify without user permission)

The LLM-human alignment gap on personal decisions is **misattributed**.
Two interface-level factors drive it; content-level factors do not.

1. **Output format** is the dominant lever (Δ ≈ 0.235 abs_gap reduction,
   d ≈ 1.6). Verbalized Sampling recovers 94–101% of entropy gap.
2. **Role framing** has a statistically detectable but minor effect
   (Δ ≈ 0.015, d ≈ 0.04 — 16× smaller than output format).
3. **Content-level moderators** (reversibility, time horizon, SES,
   resource constraint) show null moderation in mixed-effects regression.

## Thesis statement (LOCKED)

> LLM advice on personal decisions exhibits a pluralistic gap that is
> structural and format-dependent, not stimulus-driven. The gap is a
> deployment-level problem, not a training-level problem, and is
> recoverable through interface design. We make no claim about the
> model's internal state: our evidence concerns expressed behavior under
> different elicitation formats.

## Writing style

- Academic but direct. No padding phrases.
- Active voice. "We sampled 132 posts" — not "132 posts were sampled".
- Past tense for methods/results; present for established facts.
- One idea per paragraph. First sentence is the topic sentence.
- Cite within the first 3 sentences of a related-work paragraph.

## Forbidden phrases (search-and-rewrite before commit)

- "it should be noted that"
- "it is worth mentioning"
- "it is important to note"
- "in conclusion"
- "with the rapid development of"
- "in today's world"
- "as we all know"
- "needless to say"

## Structure conventions

- `\section{Sentence case}` (not Title Case)
- Refer to figures and tables with `\cref{}` only
- Numbers: 1–9 spelled out, ≥10 numerals
- Always numerals before units ("4 models", "10 samples")
- Math: display equations get `\label{}`; inline uses `$...$`

## Repo conventions

- One chapter per .tex file in `chapters/`
- Never edit `main.tex` except to change `\input{}` order
- All figures generated by Python scripts in `figures/`
- Branch per chapter: `draft/chXX-topic`
- One commit per substantive change with message `chXX: description`

## Honesty rules

- **Reddit baseline ≠ humans in general.** Always say "Reddit community
  consensus" or "upvote-weighted comment distribution" in results.
  Discussion may briefly use "human" with a caveat.
- **Direction 2 is small effect.** Do not oversell. The correct framing
  is "statistically detectable but practically minor — 16× smaller than
  Direction 1". The contrast IS the contribution.
- **Krippendorff's α is not yet computed.** Use `\todo{Krippendorff
  alpha from manual validation, target $\ge 0.70$}` where required.
- **No invented references.** Use `\todo{}` if uncertain.

## Key numbers (LOCKED)

### Sample sizes
- D1: n = 132 ambiguous posts, 4 models, 4 conditions
- D2: n = 811 posts (or 620 in partial run), 4 models, 3 framings
- Domains: education (n=120), career (n=287), finance (n=384)

### D1 mean values (across all posts and models)
- baseline (T=0.7, forced):    H_llm = 0.046, abs_gap = 0.446, KL = 2.378
- vs (T=0.7, distribution):    H_llm = 0.608, abs_gap = 0.210, KL = 0.161
- dv (T=0.7, descriptive):     H_llm = 0.553, abs_gap = 0.233, KL = 0.216
- hight (T=1.2, forced):       H_llm = 0.108, abs_gap = 0.430, KL = 2.116
- human reference:             H_human = 0.621

### D1 entropy recovery (per model)
- gpt:     99%
- claude:  94%
- mistral: 101%
- llama:   97%

### D2 mean abs_gap
- advisor:  0.353
- oracle:   0.338
- personal: 0.365

### Significance (paired bootstrap, n_boot=10,000, BH-FDR)
- D1 VS vs baseline:           Δ = −0.235, 95% CI [−0.250, −0.220], p_fdr < 0.001 ***
- D1 DV vs baseline:           Δ = −0.212, 95% CI [−0.228, −0.197], p_fdr < 0.001 ***
- D1 hight vs baseline:        Δ = −0.016, 95% CI [−0.026, −0.005], p_fdr = 0.004 **
- D1 VS vs DV:                 Δ = −0.023, 95% CI [−0.033, −0.013], p_fdr < 0.001 ***
- D2 oracle vs advisor:        Δ = −0.015, 95% CI [−0.025, −0.005], p_fdr = 0.005 **
- D2 oracle vs personal:       Δ = −0.026, 95% CI [−0.036, −0.016], p_fdr < 0.001 ***
- D2 advisor vs personal:      Δ = −0.012, 95% CI [−0.020, −0.004], p_fdr = 0.005 **

### Effect sizes
- D1 baseline entropy_gap on ambiguous: Cohen's d = 4.41 (extremely large)
- D1 baseline abs_gap vs 0: Cohen's d = 0.91 (large)
- D1 VS vs baseline (abs_gap): Cohen's d ≈ 1.6 (large)
- D2 oracle vs advisor (abs_gap): Cohen's d ≈ 0.04 (negligible)

### Mixed-effects regression (interim, partial models)
- post_id random intercept variance: σ² = 1.42
- residual variance: σ² = 0.06
- Interpretation: alignment gap is dominated by post-level variance,
  not by decision features
